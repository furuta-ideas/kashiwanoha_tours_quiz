# 柏の葉ものしりクイズ

## プロジェクト概要
柏の葉エリアに関するクイズWebアプリ（単一HTMLファイル構成）。

## 技術スタック
- HTML/CSS/JavaScript（フレームワーク不使用）
- Web Audio API（BGM・効果音）
- Google Fonts（Zen Maru Gothic）
- GitHub Pages（デプロイ先）

## ファイル構成
- `index.html` — メインアプリ（CSS・JS・base64画像すべて埋め込み、約2.8MB）
- `images/` — 元画像ファイル（参考用、アプリでは未使用）
- `.github/workflows/jekyll-gh-pages.yml` — GitHub Pagesデプロイ用ワークフロー

## 開発時の注意
- `index.html` はbase64画像データを含むため行が非常に長い。編集時はCSS/JS部分のみを対象にすること。
- ビルドツール・パッケージマネージャーは不使用。`index.html` を直接編集する。
- テストフレームワークは未導入。ブラウザで動作確認を行う。

## アプリ仕様
- 全26問からランダム10問出題
- 4択クイズ（選択肢もランダム順）
- 10秒カウントダウン（早押し得点）
- 1問最大10,000点、10問合計最大100,000点
- ランキングTop10 + 参加人数表示
- レスポンシブ対応（スマートフォン最適化、最大幅430px）
